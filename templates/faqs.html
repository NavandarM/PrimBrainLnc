Which brain regions are the source of the data used to predict lncRNAs?
We used publicly available dataset includes data from the brain regions such cerebellum (CB), striatum (STR), hippocampus (HIP), anterior cingulate cortex (ACC), dorsolateral prefrontal cortex (DPFC), ventrolateral prefrontal cortex (VPFC), premotor cortex (PMC), primary visual cortex (V1C) across different primates like human (n = 4 - 6), chimpanzee (n = 5 - 6), gorilla (n = 2) and a gibbon. ['n' represents number of samples]

How overlap is considered during search based on the genome location?
The specified length of overlap is added to start and end position of the original co-ordinates provided during search. Default value is 0.
